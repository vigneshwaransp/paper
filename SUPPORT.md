# Support Guide

Welcome to the Paper support guide! We want to make sure you get the help you need to run, build, and extend Paper.

---

## Where to Ask Questions

### 1. GitHub Issues
If you encounter a bug, have a feature request, or have questions about the codebase:
- **Bug Reports:** Open an issue using the [Bug Report Template](https://github.com/vigneshwaranspcs24-4100s-projects/paper/issues/new?template=bug_report.md).
- **Feature Proposals:** Request new features or enhancement ideas using the [Feature Request Template](https://github.com/vigneshwaranspcs24-4100s-projects/paper/issues/new?template=feature_request.md).
- **General Questions:** Ask questions about setup, configuration, or usage using the [Question Template](https://github.com/vigneshwaranspcs24-4100s-projects/paper/issues/new?template=question.md).

### 2. Discussions
For broader conversations, ideas, or showcasing your custom specialist workspace designs, please join our **GitHub Discussions** tab.

---

## Before Opening a Ticket

To help us answer your question as quickly as possible:
1. **Search existing issues:** Your question might have already been asked and answered!
2. **Provide your environment details:** Mention your Node.js version, Operating System, and Browser.
3. **Include logs/screenshots:** If your issue relates to styling or errors, copy the server console output or browser console logs.
